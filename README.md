# 📚 Sistema de Gestão de Biblioteca (POO)

Projeto desenvolvido em Python utilizando os conceitos de Programação Orientada a Objetos (POO).

## 📌 Descrição do Projeto

Este sistema simula o gerenciamento de itens de uma biblioteca, permitindo:

- Cadastro de livros
- Cadastro de revistas
- Empréstimo de itens
- Devolução de itens
- Listagem completa dos itens cadastrados

O projeto aplica os principais conceitos de POO:

- ✅ Encapsulamento
- ✅ Herança
- ✅ Polimorfismo
- ✅ Organização em múltiplos arquivos
- ✅ Versionamento com Git e GitHub

---

## 🧠 Conceitos Aplicados

### 🔒 Encapsulamento
Atributos privados utilizando `__atributo` com getters e setters.

### 🧬 Herança
A classe `Livro` e `Revista` herdam da superclasse `ItemBiblioteca`.

### 🔄 Polimorfismo
O método `exibir_detalhes()` é sobrescrito nas subclasses, apresentando comportamentos diferentes.
